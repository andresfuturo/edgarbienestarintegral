# HGW Landing Page

Landing page moderna para HGW (Health Green World) - Productos naturales y orgánicos para el bienestar.

## 🌟 Características

- Diseño moderno y responsive
- Secciones: Inicio, Productos, Comunidad, Oportunidad, Contacto
- Animaciones suaves y transiciones
- Menú móvil funcional
- Formulario de contacto interactivo
- Optimizado para GitHub Pages

## 🚀 Despliegue en GitHub Pages

### Paso 1: Subir el proyecto al repositorio existente

Abre una terminal en la carpeta del proyecto y ejecuta:

```bash
git init
git add .
git commit -m "Initial commit - HGW Landing Page"
git branch -M main
git remote add origin https://github.com/andresfuturo/edgarbienestarintegral.git
git push -u origin main
```

### Paso 2: Activar GitHub Pages

1. Ve a tu repositorio: https://github.com/andresfuturo/edgarbienestarintegral
2. Haz clic en **Settings** (Configuración)
3. En el menú lateral, haz clic en **Pages**
4. En **Source**, selecciona **Deploy from a branch**
5. En **Branch**, selecciona **main** y la carpeta **/(root)**
6. Haz clic en **Save**

### Paso 3: Esperar el despliegue

GitHub Pages desplegará tu sitio en unos minutos. La URL será:
```
https://andresfuturo.github.io/edgarbienestarintegral/
```

## 🎨 Tecnologías Utilizadas

- **HTML5** - Estructura semántica
- **Tailwind CSS** - Estilos responsive (vía CDN)
- **Font Awesome** - Iconos
- **Google Fonts** - Tipografía Poppins
- **JavaScript** - Interactividad

## 📱 Secciones

1. **Header** - Logo y navegación fija
2. **Hero** - Mensaje principal con estadísticas
3. **Productos** - Grid de 12 productos HGW
4. **Comunidad** - Beneficios de la comunidad
5. **Oportunidad** - Plan de negocio
6. **Contacto** - Formulario de contacto
7. **Footer** - Enlaces y redes sociales

## 🛠 Personalización

Para personalizar la landing page:

1. **Colores**: Modifica los colores en las clases de Tailwind (green-600, green-700, etc.)
2. **Texto**: Edita el contenido en `index.html`
3. **Imágenes**: Reemplaza los iconos de Font Awesome con imágenes reales de productos
4. **Enlaces**: Actualiza los enlaces de redes sociales y contacto

## 📧 Contacto

Para agregar tu información de contacto real:

- Busca la sección de contacto en el HTML
- Agrega tu email real en el formulario
- Actualiza los enlaces de redes sociales en el footer
